Key Features Implemented
===========================================

✅ CORE STRATEGY (PRD Requirements)
   - Trend-following scalping (M5 execution, H1 context)
   - SMA alignment + RSI momentum + Pattern triggers
   - ATR-based SL (1.5x for EURUSD)
   - 2:1 R:R Take Profit
   - Partial profits (80% at TP, 20% runner)
   - RSI divergence runner exits
   - Dynamic lot sizing
   - Daily loss limit (3%)
   - Max 1 position for EURUSD

✅ ENHANCED FEATURES (Beyond PRD)
   - Multi-session trading (London/NY/Asian/Tokyo toggles)
   - TP validation filter (aborts if crosses H1 levels)
   - SL snapping (aligns with H1 structure)
   - Dynamic risk (1-2% user selectable)
   - Commission-aware break-even
   - Comprehensive logging system

✅ SAFETY MECHANISMS
   - Lot size rounded DOWN (never over-risk)
   - SL snapping never widens beyond calculated
   - Margin checks before execution
   - Error handling on all operations
   - Graceful degradation (partial skipping if lot too small)
